## Governance changes

Foundation Status is `COMPLETE`, Version is `1.0`, and lock state is `FROZEN`. Every future implementation must comply with the locked corpus. Any Foundation change requires an explicit user request, architectural decision, named human approval, versioning, and release validation.

Correction, Clarification, Architectural Change, and Product Strategy Change are separately defined. AI may assist analysis but may not request or approve a Foundation amendment.
