## Maintaining the system

Every architectural and product decision must align with the JOHAI Constitution, North Star, Manifesto, AI Employee Principles, Product Promise, and Company Values. The Constitution remains the highest internal product-design authority.

Foundation Version 1.0 is locked. Future implementation must comply with the full Foundation Index, and changes follow the Foundation Change Policy.

Every feature sprint must update the changelog, create or update a sprint record, revise affected manual chapters, and update audience-specific documents when facts change. Every feature, UI change, AI behavior, workflow, automation, and interaction must pass the JOHAI Constitution and Decision Framework; customer-facing work must also follow **JOHAI Contextual Intelligence™**. See `AGENTS.md` and `DOCUMENTATION_RULES.md`.
